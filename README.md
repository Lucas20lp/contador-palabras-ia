 Contador de Palabras - Proyecto IA #1

Este es un proyecto básico en Python que cuenta palabras en un texto ingresado por el usuario.

 Tecnologías utilizadas
- Python
- collections (Counter)

 Funcionalidad
- Solicita un texto al usuario
- Cuenta la cantidad total de palabras
- Muestra cuántas veces aparece cada palabra

 Objetivo
Primer proyecto dentro de mi camino de 100 proyectos con Inteligencia Artificial.
